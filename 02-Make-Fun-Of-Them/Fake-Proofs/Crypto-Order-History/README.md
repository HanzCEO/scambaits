# Fake Crypto Trading Order History Generator
That is a long name huh? Well, this is how you use it:

Prerequisite:

 - Python 3.x

1. Install packages
```shell
pip3 install -r requirements.txt
```
2. Run the app
```shell
python3 generate.py
```
3. Fill out all of the forms
4. That's it!
> The result is in `result.png` file
